# AI College Assistant Chatbot

This project is a web-based AI chatbot designed to help college students
by answering common academic and administrative queries.

## Features
- Attendance rules
- Exam and result information
- Scholarship and certificate guidance
- College office information

## Technologies Used
- HTML, CSS, JavaScript
- Google Gemini API
- Firebase Hosting

## Deployment
The project is deployed live using Firebase Hosting.
